# Minimalistic-ToDo™
New Update:

Minimalistic-ToDo™ is now touch-friendly and has a cleaner interface and UI. This application now works on all mobiles in the web version. 

Minimalistic-ToDo™ is a simple and clean utility to add, clear, and manage tasks in one desktop app.

Features:

You can add tasks.

You can clear tasks.

Works well in both desktop and web applications.

Desktop app:

You can download the folder from my Github repository for either Windows or Mac:
https://github.com/RayStormz2015/Minimalistic-ToDo

Web application:

You can go to this site:
https://raystormz2015.github.io/To-do-List/

IMPORTANT: The web version is not touch-friendly for some browsers

Notes:

Creating a desktop shortcut for the application is recommended, as not creating one means you will have to download the application from the installer again to run it.

Swipe left from the task to clear it in both the web and desktop apps.
